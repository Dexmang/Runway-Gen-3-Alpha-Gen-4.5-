# Vault 007

Luxury private-sale microsite for a 2026 Aston Martin Vantage Coupe. Jet Black over Onyx Haircalf. 2,519 miles. One owner. Asking $199,007.

Built with Astro + vanilla JS. Deploys as static to `vault007.com` via Vercel. Media assets live in `../web_assets/` (resolved via `publicDir` in `astro.config.mjs`). Source of truth for copy, brand, and listing standards: `../STITCH_SECTION_COLTON.md` and the PKA Team/Colton profile.

## Commands
| Command | Action |
| :--- | :--- |
| `npm install` | Install dependencies |
| `npm run dev` | Local dev server at localhost:4321 |
| `npm run build` | Production build to `./dist/` |
| `npm run preview` | Preview the production build |
